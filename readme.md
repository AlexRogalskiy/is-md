# is-md [![Build Status](https://travis-ci.org/sindresorhus/is-md.svg?branch=master)](https://travis-ci.org/sindresorhus/is-md)

> Check if a filepath is a Markdown file


## Install

```
$ npm install is-md
```


## Usage

```js
const isMd = require('is-md');

isMd('src/unicorn.md');
//=> true

isMd('src/unicorn.png');
//=> false
```
